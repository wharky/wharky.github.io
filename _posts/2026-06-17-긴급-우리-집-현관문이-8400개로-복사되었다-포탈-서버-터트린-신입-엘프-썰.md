## 🧐 대체 무슨 일이 일어난 건가? (찰떡 비유 타임)

자, 이게 어떻게 가능하냐고요? 
쉽게 비유해 드립니다. 

여러분, 노래방에서 마이크를 스피커 바로 앞에 대본 적 있으시죠? 
그럼 어떻게 되나요? **"삐이이이이이익-!"** 하면서 귀가 터질 듯한 굉음이 나죠? 마이크로 들어간 소리가 스피커로 나오고, 그 소리가 다시 마이크로 들어가는 '무한 반복(하울링)'이 일어나기 때문입니다.

이번 사고가 딱 그 꼴이었습니다. 

우리 부서의 신입 엘프 개발자 녀석이 포탈 경로를 설정하는 라우팅 시스템을 건드렸는데, **출발지(Source)와 목적지(Destination)를 둘 다 자기 집 현관문(`localhost:8080`)으로 설정**해 둔 겁니다.

포탈 게이트웨이가 작동하는 순간:
1. 배달원이 포탈에 발을 디딤.
2. 시스템: "어? 목적지가 출발지네? 다시 거기로 보내!"
3. 배달원이 도착하는 순간, 다시 출발하는 것으로 판정됨.
4. 이 짓을 초당 40억 번 반복하며 메모리가 꽉 차버림 (**Stack Overflow**).

이걸 우리 업계 용어로 **'차원 간 무한 루프 버그'**라고 부릅니다. 

보통 컴퓨터가 이 상태가 되면 그냥 블루스크린이 뜨고 멈춥니다. 하지만 우리 서버는 '물리적 공간'을 다루는 아스트랄 서버잖아요? 시스템이 터지는 대신, 물리 법칙이 꼬이면서 진짜 배달원 아저씨의 분신이 우주 공간을 메우기 시작한 겁니다. 

이것은 마치 **변기가 막혔는데, 물이 역류하다 못해 화장실 천장을 뚫고 분수처럼 솟구쳐 오르는 상황**과 같습니다.

## 🛠️ 눈물겨운 솔루션 아키텍처: 어떻게 고쳤나?

이 괴물 같은 상황을 해결하기 위해 새벽 3시에 짠 긴급 처방전입니다.

1. **루프 감지기(IP TTL - Time to Live) 긴급 도입**
   * 배달원 아저씨의 이마에 '마법 바코드'를 붙였습니다. 포탈을 한 번 통과할 때마다 숫자가 1씩 깎입니다. 숫자가 0이 되면 포탈이 "어? 너 너무 많이 돌았어" 하고 물리적으로 튕겨내게 만들었습니다. (클럽 입구의 엄격한 가드처럼요.)
   
2. **가비지 컬렉터(Garbage Collector) 강제 기동**
   * 거실에 꽉 찬 8,000명의 복사본 배달원들을 소멸시켜야 했습니다. 진짜 배달원의 고유 영혼 ID(Session ID)를 제외한 나머지 가짜 객체(Instance)들을 한꺼번에 쓸어 담아 차원의 쓰레기통으로 던져버렸습니다. (일종의 우주적 로봇 청소기 가동이죠.)

## 💻 사고를 유발한 마법의 소스 코드 (JavaScript 버전)

자, 문제의 신입 엘프가 작성한 파멸의 코드와 제가 긴급 처방한 코드를 주석과 함께 감상해 보시죠.

```javascript
// [🚨 파멸의 원인] 신입 엘프가 작성한 코드
function openPortal(traveler, destination) {
  console.log(`${traveler.name}님이 이동 중...`);

  // 예외 처리를 안 해놔서 출발지와 목적지가 같으면 무한 루프를 돕니다.
  // "퇴근하고 싶다"는 마음으로 StackOverflow에서 대충 복사해 붙여넣은 흔적이 역력함.
  if (destination === "현관문") {
    // 앗! 목적지가 다시 현관문이네? 나 자신을 다시 호출한다! (재귀 지옥의 시작)
    return openPortal(traveler, "현관문"); 
  }
  
  return "이동 완료!";
}

// ============================================================

// [🛠️ 그롬가르의 긴급 긴급 패치 코드]
function securePortal(traveler, destination) {
  // 1. 통과 횟수 제한(TTL)을 둔다. 5번 이상 돌면 뺨을 때려서라도 멈춰 세운다!
  if (!traveler.ttl) {
    traveler.ttl = 5; 
  }

  if (traveler.ttl <= 0) {
    console.warn("🚨 [경고] 무한 루프 감지! 차원 게이트를 강제 차단합니다.");
    return "강제 퇴출 완료! 거실 바닥을 청소하세요.";
  }

  // 2. 출발지와 목적지가 같으면 애초에 문을 안 열어줌 (방화벽 작동)
  if (destination === traveler.currentLocation) {
    console.error("이 바보야! 제자리에 포탈을 열면 어떡해!");
    return "이동 실패: 너 지금 거기 있잖아.";
  }

  // 안전하게 통과할 때마다 마법 에너지를 1씩 차감
  traveler.ttl -= 1;
  return `안전하게 ${destination}으로 이동했습니다. (남은 통과 가능 횟수: ${traveler.ttl})`;
}
```
