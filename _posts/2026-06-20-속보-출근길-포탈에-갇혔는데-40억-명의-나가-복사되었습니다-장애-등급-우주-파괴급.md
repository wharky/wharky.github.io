## 🌀 원인은? "내 뒤통수를 내가 치는 무한 루프"

쉽게 설명해 드릴게요. 

원래 포탈은 **'입구(A)에서 들어가면 출구(B)로 나온다'**는 아주 심플한 슬라이드(미끄럼틀)입니다. 근데 최근에 신입 오크 개발자가 '네트워크 트래픽을 아끼겠다'며 포탈 경로 최적화 코드를 배포했습니다.

문제는 여기서 터졌습니다.

경로 탐색 알고리즘이 꼬이면서 **"만약 출구(B)가 붐비면, 입구(A)로 임시 리다이렉트하라"**는 예외 처리를 넣은 겁니다. 

이걸 우리 일상생활로 비유하자면 이렇습니다.
1. 화장실 문을 열고 들어가려는데 안에 사람이 있음.
2. 시스템이 님을 0.0001초 전의 화장실 문 앞으로 튕겨냄.
3. 근데 0.0001초 전의 '나'는 아직 문을 안 열었음.
4. 결국 **문을 열려던 '과거의 나'와 튕겨 나간 '현재의 나'가 좁은 문앞에서 충돌함.**
5. 충돌이 발생하자 시스템이 에러를 뿜으며 둘 다 다시 문 앞으로 튕겨냄.
6. 이제 문 앞에는 '나'가 4명, 8명, 16명, 32명...

이 짓을 컴퓨터는 **1초에 수십억 번** 반복합니다. 
그 결과, 단 3초 만에 3번 포탈 안에는 **40억 명의 김 아르웬 대리**가 압축 팩처럼 꽉 들어차 버린 겁니다. 이걸 개발 용어로 **'스택 오버플로우(Stack Overflow)'**라고 합니다. 그냥 컵에 물 따르라니까 우주 전체를 물바다로 만들어 버린 상황이죠.

## 🛠️ 눈물겨운 솔루션 아키텍처 (장애 복구 작전)

당장 포탈 전원을 끌 수도 없습니다. 강제 종료(`kill -9`)를 해버리면 포탈 안에 갇힌 40억 명의 김 대리가 물리적으로 압착되어 **'인간 블랙홀'**이 되어 버립니다. 지구 전체가 김 대리 인력에 끌려 들어가 멸망할 판입니다.

우리는 아주 우아하고 눈물겨운 우회로를 뚫어야 했습니다.

### 1단계: 쓰레기통 차원(Null Dimension) 임시 개방
일단 늘어나는 김 대리 복사본들을 받아줄 '임시 하수구'가 필요했습니다. 아무것도 존재하지 않는 무(無)의 차원을 급히 할당해서 포탈 출구를 그쪽으로 돌렸습니다. 
*   *비유:* 쏟아지는 물을 막을 수 없으니, 일단 안 쓰는 마당에 구멍을 파서 물길을 돌린 겁니다.

### 2단계: 가비지 컬렉터(Garbage Collector) 수동 가동
진짜 김 대리(원본)를 찾아내야 합니다. 원본은 영혼의 고유 ID(UUID)를 가지고 있습니다. 복사본들은 영혼 데이터가 비어 있는 '껍데기(섀도우 복사본)'일 뿐입니다. 
서버에 메모리 정리 프로그램을 강제로 돌려, 영혼 정보가 없는 가짜 김 대리들을 데이터 가루로 분해해 우주 먼지로 날려 보냈습니다.

## 💻 문제의 그 비극적인 코드 (JavaScript 가상 버전)

포탈 관리 시스템의 코드는 대략 이 모양 이 꼴이었습니다. 주석을 보면 개발자가 얼마나 미쳐갔는지 알 수 있습니다.

```javascript
// ⚠️ 경고: 건드리면 우주 차원축이 뒤틀림. 작성자: 신입 오크 (퇴사함)
async function transportCommuter(commuter) {
  const portalStatus = await getPortalStatus("Asgard-Line-3");

  if (portalStatus === "BUSY") {
    // [버그 지점] 출구가 막혔다고 입구로 다시 밀어 넣는 미친 짓을 저지름
    console.log(`[경고] 야! ${commuter.name} 대리님 다시 뒤로 가세요!`);
    
    // 자기 자신을 다시 호출하는 '재귀 함수'의 지옥문이 열림
    // 피드백 루프 방지(Break) 조건이 없어서 무한 복사 시작
    return transportCommuter(commuter); 
  }

  // 원래는 여기로 가야 정상 출근임
  return exitDimension("Asgard-Office-7");
}

// [긴급 패치] 소 잃고 외양간 고치기 고오급 소스코드
function emergencyFix(commuterList) {
  // 중복된 김 대리들을 학살하는 자비 없는 필터링
  const uniqueCommuter = commuterList.filter(commuter => {
    return commuter.hasRealSoul === true; // 진짜 영혼을 가진 '진짜'만 살아남는다
  });

  if (commuterList.length > 1) {
    console.warn(`[알림] 가짜 김대리 ${commuterList.length - 1}명을 퇴근(소멸) 시켰습니다.`);
  }
  return uniqueCommuter[0]; // 단 한 명만 살려서 판교로 보낸다
}
```
